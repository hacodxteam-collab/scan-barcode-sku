# Barcode Scanner Web App

## Prerequisites
- Node.js installed (v16+)

## How to Run

1. Double click `start.bat` in this folder.
   
OR

1. Open terminal
2. `cd client`
3. `npm install`
4. `npm run dev`

## Features
- **Scan Detection**: Works with handheld scanners (Keyboard Emulation).
- **Manual Input**: Fallback for damaged barcodes.
- **Offline Capable**: UI checks, mockup logic.
- **Mock Data**: Pre-loaded product database.
